= Boost.Build module for Xilinx SDK (XSDK)

:toc:

== User Manual

=== Example

The following is an example where a XSDK workspace named `ws` is
created from a hardware specification in `example.hdf` and used, along
with a C++ source file named `example.cpp` to create an application
named `example`.

Note below that the requirements of both `ws` and `example` match,
which is how Boost.Build matches.

[source, jam]
----
# Jamfile
import xsdk ;

# @todo for now this must be explicit
explicit ws ;
xsdkws ws
  : # sources
    example.hdf
  : # requirements
    <target-os>elf

    <link>static

    <architecture>arm

    <instruction-set>cortex-r5+vfpv3

    <xsdk-configuration>"sleep_time psu_ttc_3"
  : # default-build
  : # usage-requirements
    <linkflags>"-Wl,-T -Wl,lscript.ld"
  ;

exe example
  : # sources
    example.cpp

    ws
  : # requirements
    <target-os>elf

    <link>static

    <instruction-set>cortex-r5+vfpv3-d16
  ;
----
